📊 E-commerce Data Pipeline

Projeto ETL desenvolvido em Python

🧠 Problema

Processar múltiplos arquivos CSV pode gerar erros de encoding, nomes inconsistentes e dificuldades na carga para bancos de dados.

🚀 Solução

  • Detecção automática de encoding
  • Tradução automática de datasets
  • Normalização de colunas SQL
  • Pipeline automatizado de ETL
